# @jaemin/eslint-config

Personal, opinionated ESLint flat config for Jaemin's TypeScript, React, and
AI-agent-assisted coding workflows.

This repository is intentionally starting from the design spec first. The v0.1
goal is to dogfood a small ESLint 9 flat-config package against
`synchronize-tab-scrolling` before expanding the rule set.

See [`docs/superpowers/specs/2026-06-19-jaemin-eslint-config-design.md`](docs/superpowers/specs/2026-06-19-jaemin-eslint-config-design.md).
